# Smart Attendance Web Application

## Project Description
The Smart Attendance Web Application is a web-based system developed to simplify and automate the process of managing student attendance. It enables administrators to maintain student records, mark daily attendance, and view attendance details in a structured and efficient manner. The system minimizes manual effort and improves accuracy.

## Objectives
- To digitize the attendance management process
- To reduce paperwork and manual errors
- To provide quick access to attendance records
- To demonstrate basic web development concepts

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Server: XAMPP / WAMP

## Functional Features
- Secure admin login
- Add and manage student details
- Mark attendance as Present or Absent
- Store attendance data in database
- View attendance records by date and student

## Database Design
The application uses a MySQL database with the following tables:
- Admin
- Students
- Attendance

## Installation & Execution
1. Install XAMPP or WAMP server
2. Copy the project folder into the `htdocs` directory
3. Start Apache and MySQL services
4. Import the database using phpMyAdmin
5. Run the project at `http://localhost/attendance`
